About

This is a health check dashboard that will perform client side checks for a customizable list of endpoints. All checks are run client side, permitting hosting on static hosting like GitHub Pages. Fork me on GitHub customize your list.

Help

For even more debugging information open your browsers developer tools.

  1. Open the network tab to watch network requests and to see detailed information about their status.
  2. Open the console log to see logs from the application, as well as other error information provided by the web browser.
